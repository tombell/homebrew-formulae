# Homebrew Cask Formulae

A [Homebrew tap][brew-tap] for formulae and casks for some of [my tools][tombell].

## Usage

Tap this repository:

    brew tap tombell/formulae

## Formulae

Install with `brew install`.

- `coin`
- `lock`
- `lodge`
- `prettytest`
- `releasekit`
- `tldr`

## Casks

Install with `brew cask install`.

- None
